**Prerequisite**: Obtain a custom integrations YAML file and an OpenAPI Spec.


<Steps>
<Step title="Get Connected Account">
**Edit the custom integrations YAML file** and `openapiSpec`

1. Download the OpenAPI specification for Asana from the following link:
   [Asana OpenAPI Spec](https://github.com/SamparkAI/sample-openapi-spec/blob/master/Asana-OpenAPI.yaml)

2. Download the sample integrations YAML file for Asana from the following link:
   [Asana Integrations YAML](https://github.com/SamparkAI/sample-openapi-spec/blob/master/asana.yaml)


</Step>

<Step title="Add OpenAPI Spec">
Navigate to **Project Settings** and add your OpenAPI specification to import tools.

Add the name, upload the OpenAPI Spec, and the `integrations.yaml` file.

![OpenAPI Specification](/media/openapi_spec.png)
</Step>

<Step title="Verify Imported Tool">
Go back to the apps page and verify the tool that was imported.

Now you can use the tool with your agentic framework.

</Step>
</Steps>
