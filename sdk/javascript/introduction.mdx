<Steps>
<Step title="Installation Steps">
  First, ensure you have Node.js installed on your system. Then, proceed to install the `composio-core` package using your preferred package manager.
  <Tabs>
  <Tab title="npm">
    <CodeGroup>
      ```bash Console
      npm install composio-core
      ```
    </CodeGroup>
  </Tab>
  <Tab title="yarn">
    <CodeGroup>
      ```bash Console
      yarn add composio-core
      ```
    </CodeGroup>
  </Tab>
  <Tab title="pnpm">
    <CodeGroup>
      ```bash Console
      pnpm install composio-core
      ```
    </CodeGroup>
  </Tab>

  </Tabs>
</Step>
<Step title="Login on composio.dev and add a new tool">
  Head towards <a href="https://app.composio.dev" target="_blank">app.composio.dev</a> and log in to your account.
  Once logged in, navigate to the 'Tools' section to add a new tool to your project.
 ![intro](/media/js_sdk/apps.png)
</Step>

<Step title="Add a new account for the tool">
  One connector is created, add a new connected account for the tool.
  ![intro](/media/js_sdk/add_account.png)

</Step>

<Step title="Generate the API key">
  Head towards <a href="https://app.composio.dev/settings" target="_blank">app.composio.dev/settings</a>.
  Navigate to the 'API Keys' section to generate a new API key for your project.
  ![intro](/media/js_sdk/api_key.png)
</Step>
</Steps>

Keep the API key in a secure location, it will be used to authenticate your requests to the Composio API.
